# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>查看所有 Azure Web 应用资源的指标的分析数据

![Web 应用符号](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Azure Web 应用分析解决方案已移到社区支持。  
>- 该解决方案不再可从 Azure 市场获得，但可以从社区支持的 [Azure 快速入门模板](https://azure.microsoft.com/en-us/resources/templates/101-webappazure-oms-monitoring/)安装。
>- 已安装该解决方案的客户可以继续使用它进行更改。  
>- Azure 建议你使用 [Application Insights](../../azure-monitor/app/app-insights-overview.md) 监视 Web 应用程序。
