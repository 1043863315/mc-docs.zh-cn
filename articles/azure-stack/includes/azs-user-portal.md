登录到用户门户：
    - 对于集成系统部署，门户地址将根据解决方案的区域和外部域名而有所不同。 它将采用格式： https://portal.&lt;*region*&gt;.&lt;*FQDN*&gt;。
    - 如果使用的是 Azure Stack 开发工具包 (ASDK)，则门户地址为 [https://portal.local.azurestack.external](https://portal.local.azurestack.external)。
