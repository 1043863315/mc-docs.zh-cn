   - [Course 20537A:Configuring and Operating a Hybrid Cloud with Microsoft Azure Stack](https://aka.ms/azsmoc)（课程 20537A：使用 Microsoft Azure Stack 配置和操作混合云）

   - [edX：配置和操作 Microsoft Azure Stack 在线课程](https://aka.ms/AzureStackMOOC)
