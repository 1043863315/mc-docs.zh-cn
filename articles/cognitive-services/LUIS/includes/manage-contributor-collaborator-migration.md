对协作者的贡献的管理取决于应用的当前状态。

**用于创作资源迁移应用**：_参与者_在用于创作资源的 Azure 门户中使用“访问控制(IAM)”  页进行管理。 使用协作者的电子邮件地址和“参与者”  角色添加用户。 

**对于尚未迁移的应用**：所有协作者  都在 LUIS 门户中通过每个应用的“管理 -> 协作者”页进行管理。  在此页上，[添加用户的电子邮件地址](../luis-how-to-collaborate.md)。 这仅提供对此应用的创作访问权限。
