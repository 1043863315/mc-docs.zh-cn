# <a name="prebuilt-domain-reference-for-your-luis-app"></a>LUIS 应用的预构建的域参考
此参考提供有关预生成域的信息，这些信息是 LUIS 提供的意向和实体的预生成集合。

与此相反，自定义域一开始没有意向和模型。 可将任何预构建的域意向和实体添加到自定义模型中。
