# <a name="speech-sdk-api-reference"></a>语音 SDK API 参考

使用语音软件开发工具包 (SDK)，应用程序可以访问语音服务的功能，这使得开发启用了语音的软件更为容易。 当前，可通过这些 SDK 使用语音转文本、语音翻译和意向识别功能。 有关功能和支持平台的一般概述，请参阅文档[进入页](https://docs.azure.cn/zh-cn/cognitive-services/speech-service/)。
