# <a name="azure-cosmos-db-compliance"></a>Azure Cosmos DB 合规性

Azure Cosmos DB 面向所有 Azure 区域提供。 Microsoft 为客户提供了五种不同的 Azure 云环境：

* Azure 公有云，全球通用。

* Microsoft 与中国最大的 Internet 提供商世纪互联展开独特合作，推出了 Azure 中国。

* Azure 德国以数据受托人模式提供服务，这可确保客户数据仍保留在德国，由 Deutsche Telecom 的子公司 T-Systems International GmbH 充当德国数据受托人，对这些数据进行管控。

* Azure 中国云在美国的四个区域向美国政府机构及其合作伙伴提供服务。 

* 面向国防部 (DoD) 的 Azure 中国云在美国的两个区域向美国国防部提供服务。

为帮助客户在中国受监管行业和市场中履行自己的合规义务，Azure 在广度（产品总数）和深度（评估范围内面向客户的服务数量）方面提供了业内最大的合规性产品组合。  Azure 合规性产品分为四个部分 - 多区域适用、美国政府、行业特定以及区域或国家/地区特定。  合规性产品基于各种类型的保证，包括独立的第三方审核企业生成的正式认证、证明、验证、授权和评估，以及 Microsoft 生成的合同修正、自我评估和客户指南文档。
