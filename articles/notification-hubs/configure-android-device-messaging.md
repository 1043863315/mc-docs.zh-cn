# <a name="configure-android-device-messaging-adm-settings-for-a-notification-hub-in-the-azure-portal"></a>在 Azure 门户中为通知中心配置 Android Device Messaging (ADM) 设置
本文介绍如何使用 Azure 门户为 Azure 通知中心配置 Android Device Messaging (ADM) 设置。 

## <a name="prerequisites"></a>先决条件
如果你尚未创建通知中心，现在请创建一个。 有关详细信息，请参阅[在 Azure 门户中创建 Azure 通知中心](create-notification-hub-portal.md)。 

## <a name="configure-android-device-messaging"></a>配置 Android Device Messaging

以下过程提供的步骤演示了如何为通知中心配置 Amazon Device Messaging (ADM) 设置： 

1. 在 Azure 门户的“通知中心”页上，在左侧菜单中选择“Amazon (ADM)”。  
2. 输入“客户端 ID”和“客户端机密”的值。  
3. 选择**保存**。
    
   ![Azure 门户中 ADM 设置的屏幕截图](./media/notification-hubs-kindle-get-started/notification-hub-adm-settings.png)
