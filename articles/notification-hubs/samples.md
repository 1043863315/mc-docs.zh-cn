# <a name="azure-notification-hubs-samples"></a>Azure 通知中心示例
Microsoft 和第三方发布 Azure 通知中心的 SDK。 适用于 Android、iOS 和 .NET 的 Microsoft 存储库也包含每个目标平台的示例应用。

| SDK            | 发布者      |
| -------------- | -------------- |
| [.NET Server](https://github.com/Azure/azure-notificationhubs-dotnet) | Azure |

| [Apache Cordova](https://github.com/derek82511/cordova-azure-notification-hubs) | 第三方 |
| [iOS](https://github.com/Azure/azure-notificationhubs-ios) |Azure |
| [Java Server](https://github.com/Azure/azure-notificationhubs-java-backend) |Azure |

| [React Native](https://github.com/CatalystCode/react-native-azurenotificationhub) | 第三方 |
