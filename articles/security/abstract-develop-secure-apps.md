# <a name="develop-secure-cloud-applications-on-azure"></a>在 Azure 上开发安全的云应用程序
## <a name="abstract"></a>摘要
本白皮书是你在开发适用于云的应用程序时应该在软件开发生命周期的每个阶段考虑的安全问题和控制的一般指南。 在发布产品之前实现这些概念有助于构建更安全的软件。 本白皮书中提出的建议来自于我们在 Azure 安全性方面的经验以及我们客户的经验。

本白皮书旨在为构建和部署安全的 Azure 解决方案的各个层面的软件设计人员、开发人员和测试人员提供资源。
