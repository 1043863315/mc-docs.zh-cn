# <a name="azure-sql-database-learns-amp-adapts"></a>Azure SQL 数据库会自行学习和进行适应性调整

## <a name="how-azure-sql-database-learns-adapts-and-grows-with-your-application"></a>Azure SQL 数据库如何随应用程序进行学习、适应性调整和成长

Azure SQL 数据库每天管理数以十亿计的事务和数以百万计的数据库。 它的主要功能之一是 SQL 数据库会始终随着你的应用进行学习和适应性调整。 这样，可以动态的方式将性能、可靠性和数据安全性最大化&mdash;不费吹灰之力即可实现。

![学习和适应](./media/sql-database-learn-and-adapt/sql-database-learn-and-adapt.png)

## <a name="threat-detection-and-alerts"></a>威胁检测和警报  
对于威胁和异常检测，SQL 数据库内置有行为分析、实时警报、可配置的威胁策略、审核日志和检测和修复异常模式的智能方式。

[阅读文档](./sql-database-threat-detection-get-started.md)

## <a name="automatic-tuning"></a>自动优化
SQL 数据库能够让你的应用以最佳性能运行。 通过持续学习应用的模式，自适应自动调节其性能，并自动改进，无需执行任何操作&mdash;因为我们知道你也不希望如此。

[阅读文档](http://go.microsoft.com/fwlink/?LinkID=787566)

## <a name="insights-when-you-need-them"></a>在你需要时提供见解
SQL 数据库能够跟踪每个查询及其持续时间、频率和资源利用率。 基于此遥测，自动算法将优化你的数据库使其最大程度地符合你的查询。 此外，SQL 数据库会提供见解来帮助将查询优化和性能问题故障排除所需的时间降至最低。 这意味着你将获得对资源消耗量、最优先执行的查询的直接见解，并能够深入探索并获得更多详细信息。

[阅读文档](http://go.microsoft.com/fwlink/?LinkID=787567)

## <a name="no-administration-required"></a>无需管理
SQL 数据库可提供应用所需的自动管理和数据保护，以便可返回到最擅长的操作&mdash;编码。 你将自动获得备份、灾难恢复故障转移、基础结构维护、安全性和软件修补程序以及功能更新。 你不仅能够通过 SQL 数据库在后台运行而节省金钱和时间，还可以永远不再为应用停止运行的情况担心和烦恼。

[阅读文档](http://go.microsoft.com/fwlink/?LinkID=787568)
