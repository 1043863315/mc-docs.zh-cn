## <a name="prerequisite-install-the-azure-powershell-module"></a>先决条件：安装 Azure PowerShell 模块

若要执行本文中的步骤，需要 [安装和配置 Azure PowerShell](https://docs.microsoft.com/powershell/azureps-cmdlets-docs) ，始终按照说明登录到 Azure 并选择订阅。

> [!NOTE]
> 如果你没有 Azure 帐户，需要获取一个。 可以[在此处注册 1 元试用帐户](https://www.azure.cn/pricing/1rmb-trial/)。
