仅当使用 Azure 资源管理器对 Azure 存储执行管理操作时，以下限制才适用。

| 资源 | 默认限制 |
| --- | --- |
| 存储帐户管理操作数（读取） |每 5 分钟 800 次 |
| 存储帐户管理操作数（写入） |每小时 200 次 |
| 存储帐户管理操作数（列出） |每 5 分钟 100 次 |
