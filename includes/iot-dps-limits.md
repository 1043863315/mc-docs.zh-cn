下表列出了适用于 Azure IoT 中心设备预配服务资源的限制。

| 资源 | 限制 |
| --- | --- |
| 每个 Azure 订阅的最大设备预配服务数 | 10 个 |
| 最大登记数 | 500,000 |
| 最大注册数 | 500,000 |
| 最大登记组数 | 100 |
| 最大 CA 数 | 25 |

> [!NOTE]
> 可联系 [Azure 支持部门](https://www.azure.cn/support/contact/)，要求增加订阅中的实例数。

超过以下配额时，设备预配服务将限制请求：

| 限制 | 每单位值 |
| --- | --- |
| 操作 | 200/分钟/服务 |
| 设备注册数 | 200/分钟/服务 |
| 设备轮询操作 | 5/10 秒/设备 |
