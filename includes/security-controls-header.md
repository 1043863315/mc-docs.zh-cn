安全控制是促使 Azure 服务能够防范、检测和响应安全漏洞的一种服务质量或功能。

对于每项控制，我们使用“Yes”或“No”来指示它当前是否用于该服务，对于不适用于该服务的控制为“N/A”。 我们还可能会提供有关属性的更多信息的注释或链接。
