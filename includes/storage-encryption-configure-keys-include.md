Azure 存储支持使用 Azure 管理的密钥或客户管理的密钥进行静态加密。 使用客户管理的密钥，你可以创建、轮换、禁用和撤销访问控制。

使用 Azure Key Vault 管理你的密钥并审核你的密钥使用情况。 你可以创建自己的密钥并将其存储在密钥保管库中，也可以使用 Azure Key Vault API 来生成密钥。 存储帐户和 Key Vault 必须在同一个区域中，但可以在不同的订阅中。 有关 Azure 密钥保管库的详细信息，请参阅[什么是 Azure 密钥保管库？](../articles/key-vault/key-vault-overview.md)
