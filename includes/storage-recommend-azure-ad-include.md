> [!TIP]
> Azure 存储支持使用 Azure Active Directory (Azure AD) 授权对 Blob 和队列存储的请求。 与共享密钥授权相比，使用 Azure AD 返回的 OAuth 2.0 令牌对用户或应用程序授权具有更高的安全性和易用性。 使用 Azure AD 时，不需将帐户访问密钥与代码存储在一起，因此没有潜在的安全漏洞风险。
>
> Azure 建议尽量对 Azure 存储应用程序使用 Azure AD。 有关详细信息，请参阅[使用 Azure Active Directory 授予对 Azure Blob 和队列的访问权限](../articles/storage/common/storage-auth-aad.md)。
