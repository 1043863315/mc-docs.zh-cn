## <a name="supported-ssh-key-formats"></a>受支持的 SSH 密钥格式

Azure 目前支持最小长度为 2048 位的 SSH 协议 2 (SSH-2) RSA 公钥-私钥对。 不支持其他密钥格式（如 ED25519 和 ECDSA）。
