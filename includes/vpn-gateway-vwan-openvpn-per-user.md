### <a name="open-the-mfa-page"></a><a name="mfa"></a>打开 MFA 页

1. 登录到 Azure 门户。
2. 导航到“Azure Active Directory”->“所有用户”  。
3. 选择“多重身份验证”  以打开“多重身份验证”页。

    ![登录](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a> 选择用户

1. 在“多重身份验证”  页上，选择要为其启用 MFA 的用户。
2. 选择“启用”  。

    ![Select](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
